To run this project locally:

```bash

git clone https://github.com/Kaushikkkkkk250106/Kaushik-portfolio.git

# Navigate into the project folder

# Open in browser
open index.html
```

### 🌐 International Debt Analysis Platform *(Final Year Project)*

A comprehensive analytics platform that ingests World Bank and IMF debt datasets, performs statistical analysis, and generates interactive visualisations.

- Multi-country debt trend analysis with time-series visualisation
- Debt-to-GDP ratio benchmarking across 50+ economies
- Interactive Power BI dashboard for stakeholder reporting
- **Stack:** Python, Pandas, Matplotlib, Seaborn, SQL, Power BI

### 🛒 Retail Data Cleaning & Analysis

End-to-end data wrangling on retail transaction data with automated pipelines for missing-value imputation, outlier detection, and business insights.

- Z-Score-based outlier detection & removal
- Top-10 customer segmentation analysis
- **Stack:** Python, Pandas, NumPy, Seaborn
